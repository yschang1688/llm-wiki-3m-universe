# BERT

> [!abstract] **TL;DR**:
> 基於 Transformer Encoder 的雙向編譯模型，透過「遮蔽語言模型」與「下一句預測」習得強大的語境感應詞嵌入能力，並展現出驚人的跨語言遷移與跨學科應用潛力。

| 欄位 | 內容 |
| --- | --- |
| 類別 | LLM 預訓練模型 |
| 提出年 | 2018 |
| 父頁 | `[[Transformer架構]]` |
| 主要應用 | 語境感應詞嵌入、零樣本跨語言遷移、跨學科序列分析 |
| 別名 | Multilingual BERT, 跨學科 BERT |

## 重點
- **【核心發現】**：雙向上下文理解與大規模自監督預訓練，使模型能捕捉詞彙深層語義，並形成具備跨語言與跨學科通用性的表示空間。
- **語境感應詞嵌入 (Contextualized Word Embedding)**：解決傳統 Word2Vec 或 GloVe 無法處理一詞多義（Polysemy）的問題。例如「銀行」在金融與河岸語境下會獲得不同的向量表示。
- **Transformer Encoder 架構**：完全基於 Self-attention 機制，能同時觀察句子中所有符號的關聯性，不再受限於 RNN 的序列性計算，有利於捕捉長距離依賴。
- **雙重預訓練任務**：
    - **Masked LM (填空題)**：隨機遮蔽（Mask）15% 的輸入字元，迫使模型利用上下文預測該字，藉此學習語法與語義。
    - **Next Sentence Prediction (NSP)**：預測兩個句子是否具備連續性，強化模型對段落邏輯與連貫性的理解。
- **Fine-tuning 範式**：預訓練完成後，僅需針對特定下游任務加上簡易的線性層並微調，即可在各類 NLP 基準測試中達到頂尖效果。
- **跨語言與跨領域遷移**：即使僅用英文資料微調，模型也能解決中文 QA 問題；且在人類語言上預訓練的權重，可有效提升 DNA 或音樂分類任務的表現。

## 細節

### 基礎資訊
- **開發者**：Google AI
- **核心架構**：Transformer Encoder
- **訓練目標**：Masked LM (MLM), Next Sentence Prediction (NSP)

### 詞嵌入演進對照表

| 技術 | 代表模型 | 核心特性 | 缺點 |
| --- | --- | --- | --- |
| **1-of-N Encoding** | N/A | 每個詞為獨立編碼 | 維度過大，無法表達關聯性 |
| **Static Embedding** | Word2Vec, GloVe | 每個 Type 固定一個向量 | 無法解決一詞多義問題 |
| **RNN-based Contextual** | ELMO | 利用雙向 LSTM 提取特徵 | RNN 計算效率低，長程依賴有限 |
| **Self-attention Contextual** | **BERT** | 全雙向 Transformer 結構 | 預訓練計算成本高 |

### 核心預訓練機制

#### 1. Masked Language Model (MLM)
BERT 在輸入序列中隨機選擇 15% 的 Token 進行預測任務：
- 其中 80% 被替換為 `[MASK]`。
- 10% 被替換為隨機的錯誤 Token。
- 10% 保持不變。
這迫使模型必須理解雙向的上下文資訊才能準確推論。

#### 2. Next Sentence Prediction (NSP)
為了捕捉句子間的關係，BERT 會接收成對的句子 [A, B]：
- 50% 的機率 B 是 A 的下一句（IsNext）。
- 50% 的機率 B 是隨機抽取的無關句子（NotNext）。
透過起始處的特殊符號 `[CLS]` 輸出的向量進行二進位分類。

### 跨語言能力 (Cross-lingual)
- **零樣本轉移 (Zero-shot Transfer)**：僅使用英文 QA 資料微調的 Multi-BERT，能自動具備解決中文 QA 的能力。這顯示模型將不同語言中意義相同的符號映射至相近的向量空間。
- **語言差異向量**：中英文表示之間存在一個可平移的「差異向量」，可用於粗略翻譯（如 fish + 差距 ≈ 魚）或修正跨語言表現落差。

### 跨領域與跨學科應用 (Cross-disciplinary)
- **通用分類器**：在人類語言上預訓練的模型，可應用於 DNA 序列、蛋白質結構及音樂特徵分類，且表現優於從頭訓練的模型。
- **人造資料實驗**：研究發現，使用具備週期性結構或經過 Shuffle 的數字序列進行預訓練，對 NLP 任務仍有顯著幫助（可達英文預訓練進步量的 75%），證明模型習得的「長範圍依賴性」處理能力是核心。

### 端到端語音問答 (Speech QA)
透過將語音預訓練模型（如 HuBERT）提取的 Token 隨機映射至英文單字，再丟給文字 BERT 進行初始化，可實現不依賴語音辨識（ASR）的端到端語義理解。

### 特殊符號 (Special Tokens)
- **`[CLS]`**：位於序列最前端，用於整句/整對句子的特徵代表。
- **`[SEP]`**：用於分隔兩個句子。
- **`[MASK]`**：用於填空預訓練任務。
