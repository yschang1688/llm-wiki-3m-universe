# 參數高效微調與低資源學習

> [!abstract] **TL;DR**:
> 探討在計算資源有限或標註資料稀缺的場景下，如何有效利用大型預訓練模型。涵蓋 PEFT 技術（LoRA, Adapter 等）及低資源學習範式（Prompting, In-Context Learning）。

| 欄位 | 內容 |
| --- | --- |
| 類別 | LLM 微調 / 遷移學習 |
| 主要應用 | `[[LoRA]]`, `[[Adapter]]`, `[[Prefix Tuning]]`, `[[提示工程]]` |
| 別名 | PEFT, Low-Resource Learning |

## 重點
- **低資源學習 (Low-Resource Learning)**：
    - **提示學習 (`[[Prompting]]`)**：透過設計提示模板（Template）將任務轉換為自然語言，激發模型預訓練知識。
    - **情境學習 (In-Context Learning)**：在輸入中提供少數示範（Examples）來引導模型完成任務。
    - **半監督學習**：利用 Prompt 對大量未標註資料產生假標籤（Pseudo Label）再進行微調。
- **參數高效微調 (PEFT)**：僅訓練原模型 0.1% - 10% 的參數，大幅降低儲存與運算開銷。主要流派包含：
    - **序列式**：`[[Adapter]]`。
    - **平行式**：`[[LoRA]]`。
    - **注意力優化**：`[[Prefix Tuning]]`。
    - **輸入層優化**：Soft Prompts。

## 細節
### 實際應用中的部署挑戰
1. **儲存爆炸**：傳統微調需為每個下游任務備份整個模型，導致儲存空間隨任務數呈倍數成長。
2. **推論效率**：超大規模模型（如 GPT-3）在部署時面臨推論延遲高的問題，可透過「動態推論 (Dynamic Inference)」在簡單樣本上提早結束計算來優化。

### 技術比較表
| 技術 | 插入位置 | 推論延遲 | 參數量 |
| --- | --- | --- | --- |
| `[[LoRA]]` | 線性層旁 (平行) | 無 | 極低 |
| `[[Adapter]]` | 層間 (序列) | 略有增加 | 低 |
| `[[Prefix Tuning]]` | Attention K/V | 有 | 極低 |
| **Soft Prompts** | 輸入 Embedding | 無 | 最低 |

## 相關概念
- `[[LoRA]]`、`[[Adapter]]`：具體技術細節。
- `[[動態推論]]`：減少推論計算量的策略。
- `[[預訓練對齊效益與界限]]`：關於對齊階段的討論。
