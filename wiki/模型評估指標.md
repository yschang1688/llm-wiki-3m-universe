# 模型評估指標

> **TL;DR**：模型評估指標依任務類型分為分類（Accuracy／Precision／Recall／F1／ROC-AUC，源自混淆矩陣）與迴歸（MSE／RMSE／MAE／R²）兩大族；指標選擇需依業務成本（假陽性 vs 假陰性）決定，並搭配 [[交叉驗證]] 確保估計穩健。

## 重點

- **【核心發現】**：**沒有萬用指標——類別不平衡時 Accuracy 高度誤導，必須用 Precision／Recall／F1／ROC-AUC 補強；指標選錯等於量錯目標。**
- **分類指標**：源自混淆矩陣（TP／TN／FP／FN）四元組——Accuracy、Precision、Recall、F1-Score、ROC-AUC。
- **迴歸指標**：MSE（懲罰大誤差）、RMSE（同單位、易解讀）、MAE（抗離群）、R²（解釋變異比例）。
- **驗證穩健性**：單一 Holdout 易受切分偶然性影響，應搭配 [[交叉驗證]]（K-fold／Stratified K-fold）取得更可信的估計。
- **業務成本決定主指標**：誤判正類成本高（垃圾郵件）→ 重 Precision；漏判成本高（疾病篩檢）→ 重 Recall。

## 細節

### 分類指標公式速查
| 指標 | 公式 | 適用情境 |
|---|---|---|
| Accuracy | (TP+TN) / 全部 | 類別平衡時 |
| Precision | TP / (TP+FP) | 誤判正類成本高 |
| Recall | TP / (TP+FN) | 漏判成本高 |
| F1-Score | 2·P·R / (P+R) | Precision／Recall 折衷，不平衡資料首選 |
| ROC-AUC | ROC 曲線下面積 | 對閾值穩健，衡量整體排序能力 |

### 迴歸指標對照
| 指標 | 特性 | 適用情境 |
|---|---|---|
| MSE | 平方誤差平均，懲罰大誤差 | 誤差分佈穩定 |
| RMSE | √MSE，與目標變數同單位 | 需直觀解讀誤差量級 |
| MAE | 絕對誤差平均 | 含離群值的資料 |
| R² | 1 − SS_res/SS_tot，解釋力指標 | 模型解釋力比較 |

### IPAS 中級科目 3 §5.3 視角：評估雙軸與產業實務指標
IPAS《AI 應用規劃師（中級）》學習指引 §5.3「模型訓練、評估與驗證」將評估拆為「指標選擇」與「驗證策略」雙軸，並補充兩項本頁表格未涵蓋的實務指標。
- **PSI（Population Stability Index）**：衡量訓練資料與線上資料的特徵分佈差異，是預測模型效能下滑最具領先性的監控指標；與 KL 散度同為資料漂移（Data Drift）偵測工具。
- **模型漂移（Model Drift）vs 概念漂移（Concept Drift）**：前者指預測準確度隨時間變差、需重新調整模型；後者指資料背後的潛在關係發生變動，導致原有預測邏輯失效——兩者診斷方法不同，但都需持續監控觸發重訓（Auto-retraining）。
- **驗證策略**：訓練／驗證／測試三集切分不可互滲；資料量有限時以 [[交叉驗證]] 的 K-fold／Stratified K-fold（分類任務保持類別比例）提升評估穩健性。
- **考點對映**：「不平衡資料應用何指標」（F1、ROC-AUC）、「PSI 的用途」（偵測特徵分佈飄移、觸發模型重訓）、「Recall 定義」（真實正類中被正確預測為正的比例）。
