# 基礎模型

> **TL;DR**：基礎模型 (Foundation Model) 是以[[自我監督式學習]]在大規模資料上預訓練、可遷移至多種下游任務的大型神經網路；2021 年由 Stanford CRFM 正式命名，[[BERT]]、GPT、CLIP、LLaMA 均屬此類。

| 欄位 | 內容 |
|---|---|
| 類別 | AI 範式 / 預訓練模型 |
| 提出年 | 2021 (Stanford CRFM 命名) |
| 主要應用 | LLM、多模態、跨任務遷移 |
| 父頁 | [[深度學習]] |
| 子頁 | [[BERT]]、[[Transformer架構]] |
| 難度 | ★★★☆☆ |
| 別名 | Foundation Model、預訓練基底模型 |

## 重點

- **【核心發現】**：**「先在巨量無標註資料上自監督預訓練、再針對下游任務微調」的範式轉移，使單一模型可服務數百種任務，徹底改寫了 AI 研發與商業化路徑。**
- **三大特徵**：
    - **規模**：億級至兆級參數、TB 級訓練資料。
    - **通用性**：同一模型可遷移至翻譯、摘要、問答、生成等多種任務。
    - **突現能力 (Emergence)**：規模達到臨界點後出現訓練時未設計的能力（in-context learning、reasoning）。
- **與「預訓練模型」差異**：基礎模型強調「通用底座」與「下游任務的多樣性」，並涵蓋多模態（不限於 NLP）。

## 細節

### 代表性基礎模型譜系

| 模態 | 代表模型 | 預訓練任務 |
|---|---|---|
| 文字 | [[BERT]]、GPT-3/4、LLaMA、Claude | MLM、Autoregressive |
| 影像 | ViT、DINOv2、MAE | 對比、補全 |
| 多模態 | CLIP、Flamingo、GPT-4V | 圖文對比、interleaved |
| 語音 | wav2vec 2.0、Whisper | 對比、序列轉錄 |

### 下游使用模式
1. **Fine-tuning**：以下游資料微調全部或部分參數。
2. **Prompt-based**：透過 prompt 啟動 in-context learning，無需更新參數。
3. **Adapter / LoRA**：凍結基底，新增小量可訓練參數。
4. **RAG**：以檢索外接知識，補基礎模型參數記憶不足。

### 商業與倫理議題
- 訓練資料來源、版權、隱私。
- 能力外溢與濫用風險。
- 算力集中與生態壟斷。
